Ordered List

In HTML, an ordered list is used to display a list of items in a specific order, typically numbered. The items are listed in a sequence, and each item is represented by the <li> (list item) tag. The ordered list itself is created using the <ol> tag.

Customizing the Ordered List:
We can customize the appearance of the numbering by using the 'type' attribute of the <ol> tag. Here are some options for the type attribute:

  • Default (1, 2, 3, …):
  • Lowercase letters (a, b, c, …):
  • Uppercase letters (A, B, C, …):
  • Roman numerals (i, ii, iii, …):
  • By default, an ordered list uses numbers starting from 1. We can also change the starting number using the 'start' attribute.
