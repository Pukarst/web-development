The id attribute

The id attribute specifies a unique id for an HTML elements. The value of the id attribute must be unique within the HTML document. The id attribute is used to point a specific style declaration in a style sheet. It is also used by JavaScript to access and manipulate the elements with the specific id.
The syntax for id is: writing a hash character(#), followed by id name. Then ,define the CSS properties within the curly braces{}.

NOTE

  1. The id name is case sensitive.
  2. The id name must contain at least one character.
  3. The id name can't start with a number, and must not contain any spaces,tabs.
