Unordered List

In HTML, an unordered list is used to display a list of items in no particular order. The items are typically preceded by a bullet point or the other marker rather than number. To create an unordered list, the , <ul> (unordered list) element is used, and each item within the list is placed inside <li> (list item) tags.


The example of the unordered list are:

  • Pencil
  • Pen
