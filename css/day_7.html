CSS Overflow

The overflow property in CSS controls what happens when the content is too big to fit inside its container (like a div, box, section, etc). This is a useful property because imagine that the user has a box with a height of 200px, but the content inside it is 300px. Does the user want to cut it off, scroll, or just let it overflow? That's what the overflow property lets the user decide.
