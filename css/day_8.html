CSS Float

The float property in CSS was originally designed for floating images within text, like in newspapers or magazines, where text wraps around images. But it quickly became a popular layout tool to create multi-column layouts before CSS Flexbox and Grid came along.

What's Next

The topic for the next day is 'CSS inline-block'. The 'inline-block' is a display value you can assign to HTML elements that combines features of both inline and block elements.

The float property can have one of the following values:

  • left - The element floats to the left of its container
  • right - The element floats to the right of its container
  • none - The element does not float.This is default
  • inherit - The element inherits the float value of its parent
