CSS Background

In CSS, the background property is used to define the background effects for the elements, such as color,images,positioning,repeat behavior and more.


CSS Background-Color

  • The background-color property specifies the background color of an element.
  • The opacity property specifies the opacity/transparency of an element.
  • It can take a value from 0.0-1.0.
  • When using the opacity property to add the transparency to the background of an element, all of it's child elements inherit the same transparency.
  • If we don't want to apply opacity to child elements then we can use RGBA color values.
  • An RGBA color value is specified with: rgba(red, green, blue, alpha). The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

CSS Background Image

  • The background-image is the property that specifies an image to use as the background of an element.
  • By default, the image is repeated so it covers the entire element horizontally and vertically.
  • The background-position property is used to specify the position of the background image.

CSS Background Shorthand

  • To shorten the code, it is also possible to specify all the background properties in one single property. This is called a shorthand property.
  • When using the shorthand property the order of the property values is:
    • background-color
    • background-image
    • background-repeat
    • background-attachment
    • background-position
  • We do not use the background-attachment property in the shorthand, as it does not have a value.
