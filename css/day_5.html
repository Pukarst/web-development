CSS BORDER

Solid Border Box --> A single, continuous solid line around the element which is simple and commonly used for clear visual boundaries.
Dashed Border Box --> A series of short dashes instead of a solid line and gives a more casual or stylized look.
Dotted Border Box --> A series of round dots and is often used for decorative or non-intrusive borders.
Double Border Box --> Displays two parallel solid lines with space between them and often used for emphasis or formal design.
Groove Border Box --> Creates a 3D grooved effect that looks like the border is carved into the page and shading depends on the color provided
Rounded Corners Border Box --> Uses a solid border but adds rounded corners and is visually softer than sharp-cornered boxes.
Custom Sides Border Box --> Useful for creative or asymmetric designs.
