CSS Selectors

CSS selectors are the patterns that are used to select and style the HTML elements on the web page. They tell the browser which elements that the user wanted to apply user's CSS style.

Why are CSS selectors important?

The CSS selectors are important because they control where the user's styles go. Without selectors, CSS wouldn't know what to style. It would be like trying to decorate a room without knowing which room we are in.


Selector Example Example description
Universal * select all elements
Type/element p,h1,div,span,etc select all element of that type
Class .menu select element with class name='menu'
ID #header select element with id='header'
Grouping h1,p select both <h1> and <p> element
Descendant div p select all <p> inside <div>
Child ul > li select only direct <li> children of <ul>
Adjacent Sibling h1+p select first <p> immediately after an <h1>
Attribute input[type='text'] select <input> with a type attribute of 'text'
Pseudo-class a:hover select an <a> when the mouse is hovering over it
Pseudo-element p::first-line select the first line of a <p>
